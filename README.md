# Flashcard Quiz App

A simple and interactive flashcard quiz web application built using **HTML, CSS, and JavaScript**. Users can create flashcards, review answers, edit or delete cards, and test themselves in quiz mode.

## Features

* Add flashcards with questions and answers
* Edit and delete flashcards
* Show / hide flashcard answers
* Start quiz mode based on created flashcards
* Submit answers and view final score
* Restart quiz anytime
* Flashcards stored in local storage

## Tech Stack

* HTML
* CSS
* JavaScript

1. Download or clone the repository
2. Open `index.html` in your browser
